# 数据接口密钥获取与配置指南

> 适用文件：`./.env`
>
> 当前需要配置的字段：
>
> - `TUSHARE_TOKEN`
> - `JOINQUANT_USERNAME`
> - `JOINQUANT_PASSWORD`
> - `EASTMONEY_COOKIE`

---

## 1. TUSHARE_TOKEN（优先配置）

### 获取步骤
1. 打开 `https://tushare.pro` 并注册/登录。
2. 进入个人中心（头像菜单中的 Token/API Token）。
3. 复制 Token 字符串。
4. 写入 `.env`：
   - `TUSHARE_TOKEN=你的真实token`

### 说明
- 部分接口需要积分权限。
- 本项目脚本检测到有效 token 后，会优先使用 Tushare 抓取行情；未配置则自动降级到公开源。

---

## 2. JOINQUANT_USERNAME / JOINQUANT_PASSWORD

### 获取步骤
1. 打开 `https://www.joinquant.com` 注册/登录。
2. 账号（手机号/邮箱）填写到 `JOINQUANT_USERNAME`。
3. 登录密码填写到 `JOINQUANT_PASSWORD`。

### 说明
- 当前主抓取脚本尚未强依赖 JoinQuant，可先留占位；后续接入时再启用。

---

## 3. EASTMONEY_COOKIE

### 获取步骤
1. 在浏览器登录东方财富相关页面。
2. 按 `F12` 打开开发者工具，进入 Network。
3. 刷新页面，选任意请求。
4. 在 Request Headers 复制 `cookie` 全字符串。
5. 写入 `.env`：
   - `EASTMONEY_COOKIE=完整cookie字符串`

### 说明
- Cookie 会过期，需要定期更新。
- 不要把 Cookie 公开或上传到仓库。

---

## 4. .env 示例

```dotenv
# 数据接口密钥（占位符）
TUSHARE_TOKEN=your_tushare_token_here
JOINQUANT_USERNAME=your_joinquant_username_here
JOINQUANT_PASSWORD=your_joinquant_password_here
EASTMONEY_COOKIE=your_eastmoney_cookie_here

# 抓取稳定性参数（可选）
HTTP_RETRIES=2
EVENT_MAX_PAGES=6
MARKET_MAX_SYMBOLS=30
MARKET_PUBLIC_RANGE=2y
```

---

## 5. 安全与提交建议

- `.env` 应加入 `.gitignore`，避免提交到远程仓库。
- 密钥泄露后立即重置。
- 比赛账号建议与个人生产账号分离。

---

## 6. 配置后自检

配置完成后执行：
- 运行 `src/01_ingest_events.py`
- 检查输出：
  - `data/raw/events_raw.csv`
  - `data/raw/market_daily.csv`
  - `data/raw/trading_status.csv`

若 `TUSHARE_TOKEN` 生效，日志会出现：
- `检测到TUSHARE_TOKEN，优先抓取Tushare日线...`
